# Public Installer Entry Points

## Purpose

Contains public entry scripts used via curl | bash. Responsible for OS detection and dispatching to distribution-specific installers.

## Role in WCP360 Installer Architecture

This directory is part of the WCP360 professional modular installer system.
It follows a structured, maintainable, and production-ready architecture inspired by enterprise control panels.

## Notes

- Do not remove this directory.
- All files inside must remain version-controlled.
- Changes here affect installation logic and system provisioning.
